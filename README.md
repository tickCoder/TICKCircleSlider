# TICKCircleSlider
A circle slider for iOS.
适用于iOS的圆形Slider/Progress.

# 引入方式
pod 'TICKCircleSlider', :tag =>'0.0.4', :git => 'https://github.com/tickCoder/TICKCircleSlider'
